Introduction

NeoAdmin is a modern, responsive admin dashboard template built with Bootstrap 5.3, TypeScript, and SASS (SCSS). It abandons jQuery in favor of a modern vanilla JS/TS approach, bundled with Vite for blazing fast performance.

Directory Structure

│
├── docs             <- Production ready files (HTML/CSS/JS)
│   ├── assets
│   │   ├── css       <- Compiled SCSS
│   │   ├── js        <- Compiled Bundle (neoadmin.js)
│   │   └── images
│   └── *.html
└── src              <- Source Code
    ├── scss         <- SASS source files
    │   ├── neoadmin
    │   └── ...
    └── ts           <- TypeScript source files (Logic)
        └── neoadmin.ts

Build System (Vite)

This project uses Vite for development and bundling. Node.js is required.

1. Installation

Install dependencies via npm:

npm install
2. Development Server

Start a local dev server with hot reload:

npm run dev
3. Production Build

Compile TypeScript and SCSS into `docs/assets`:

npm run build:universal

For Vue.js compatible builds (modules):

npm run build:vue

Customization

Styles (SCSS)

All styles are located in `src/scss`. The main entry point is `neoadmin.scss`.

  • Variables: Edit `src/scss/neoadmin/_vars.scss` to change colors, fonts, and sizes.
  • Bootstrap: We modify Bootstrap via `_customize-bootstrap.scss` before importing it.

Localization (i18n)

NeoAdmin supports automatic localization for libraries like DataTables.

To set the language, simply change the lang attribute in your HTML tag:

<html lang="es">   <!-- Spanish -->
<html lang="en">   <!-- English -->
<html lang="fr">   <!-- French -->

Supported libraries seamlessly adapt to this setting on page load.

Features & Persistence

NeoAdmin includes built-in functionality for user preferences which are automatically persisted (saved) using localStorage or cookies. These settings remain active even after refreshing the page.

  • Dark/Light Mode: Toggles the theme between light and dark modes.
  • Full Screen: Toggles the browser's full-screen mode.
  • Font Size: Adjusts the global font size (Small, Normal, Large).

NeoAdmin Objects

The global NeoAdmin object provides easy access to notifications and alerts.

1. Notifications (Toasts)

Show a corner notification (Toast) using `NeoAdmin.notify()`.

// Types: 'success', 'danger', 'info', 'warning'
NeoAdmin.notify('Update successful!', 'success');
2. Alerts (Modals)

Show a modal alert using `NeoAdmin.alert()` or `NeoAdmin.swal()`.

NeoAdmin.alert('Good Job!', 'Operation completed.', 'info');
3. Programmatic Bootstrap Toasts

Trigger a standard Bootstrap 5 toast by its ID.

NeoAdmin.showToast('liveToastId');
